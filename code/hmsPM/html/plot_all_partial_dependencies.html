plot_all_partial_dependencies {hmsPM}R Documentation

Plot all partial dependencies

Description

Plot list of all partial dependencies

Usage

plot_all_partial_dependencies(df_partialdep, df_data, feature_names,
  target_name = "target", individual_flag_name = NULL, ylim = NULL,
  colors = NULL, min_width = 0.2, show_1_legend = TRUE,
  df_partialdep_cv = NULL, add_cv_lines = FALSE,
  add_cv_conf_int = FALSE, verbose = TRUE)

Arguments

df_partialdep

Data frame with structure as created by calc_partial_dependencies.

df_data

Feature and target data.

feature_names

Names of features to plot.

target_name

Target name.

individual_flag_name

Optional column in df_data comprising 0 or 1 and which marks individual records to be highlighted in plots, i.e. black border or black dot.

ylim

Y-axis limits. See also lims.

colors

Colors for target levels. Used in (multi-)class performance plots.

min_width

Minimum width of bars for categorical variables. Only relevant for (multi-)class targets.

show_1_legend

Show legend only on first plot.

df_partialdep_cv

(Optional) Data frame with structure as created by calc_partial_dependencies but with additional column run distinguishing cross-validation runs. Needs to be passed if either add_cv_lines or add_cv_conf_int is TRUE.

add_cv_lines

Add lines per cross validation run to plot?

add_cv_conf_int

Add confidence intervals derived from cross validation?

verbose

(Logical) Show messages?

Value

List with all partial dependecy plots.
