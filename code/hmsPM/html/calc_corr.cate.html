calc_corr.cate {hmsPM}R Documentation

Calclulate Contingency Matrix

Description

Calclulate Contingency Matrix

Usage

calc_corr.cate(df, cutoff, show_n_levels = TRUE,
  n_level_prefix = "#Levs: ", verbose = TRUE)

Arguments

df

Data frame with categorical variables.

cutoff

Lower cutoff level. Value in [0,1]. Remove rows and columns in which no correlation value is above the cutoff value.

show_n_levels

(Logical) Show number of levels after variable names?

n_level_prefix

Prefix for number of levels.

verbose

(Logical) Show messages?

Value

Contingency matrix.

Examples

n_rows = 1e3
df_test = data.frame(
  c1 = letters[1:10][sample(1:10, size = n_rows, replace = T)],
  c2 = letters[1:10][sample(1:10, size = n_rows, replace = T)],
  c3 = letters[1:10][sample(1:10, size = n_rows, replace = T)]
)
calc_corr.cate(df_test, 0)
