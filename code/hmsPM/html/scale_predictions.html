Scale Predictions

Description

Calculate predictions from probability of sample data and the corresponding base probabilities (classification).

Usage

scale_predictions(yhat, b_sample = NULL, b_all = NULL)

Arguments

yhat

Prediction values.

b_sample

Priori probabilities of sampled data.

b_all

Priori probabilities of all data.
