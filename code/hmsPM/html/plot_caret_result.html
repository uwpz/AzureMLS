Plot Caret Result

Description

Plot Caret Result

Usage

plot_caret_result(fit, metric = "AUC", x = NULL, color = NULL,
  linetype = NULL, shape = NULL, facet = NULL, ylim = NULL,
  errorbar = FALSE)

Arguments

fit

Caret fit.

metric

Metric to plot.

x

Variable (name) used for x-axis.

color

Variable (name) used for color grouping.

linetype

Variable (name) used for linetype grouping.

shape

Variable (name) used for shape grouping.

facet

Character value with facet formula.

ylim

Y-axis limits. See also lims.

errorbar

Plot error bars (in case of crossvalidated caret fit) if TRUE.

Value

ggplot2 object
