cate_encoding.self_topn {hmsPM}R Documentation

Self Encoding of Top N Levels

Description

Self encoding of the top N most frequent feature levels (n_top_levels).

Usage

cate_encoding.self_topn(x, n_top_levels = 20, enc_nas = TRUE,
  na_value = NULL, other_value = NULL)

Arguments

x

Feature values to be encoded.

n_top_levels

Maximum number of levels.

enc_nas

(Logical) Encode missing values? Default = TRUE.

na_value

(Optional) Replacement value for NAs. The default value for metr features is 0.

other_value

(Optional) Replacement value for unknown/other values to be encoded later.

Value

Named vector with encoding for feature values.

Examples

cate_encoding(letters[c(1:3,2:3,3)], method = "self_topn", n_top_levels = 2)
