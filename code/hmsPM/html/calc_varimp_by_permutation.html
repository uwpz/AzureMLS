calc_varimp_by_permutation {hmsPM}R Documentation

Calculate feature importances

Description

Calculte feature importances by permutation argument.

Usage

calc_varimp_by_permutation(df_data, fit, formula_string, sparse = TRUE,
  metric = "AUC", features = NULL, b_sample = NULL, b_all = NULL)

Arguments

df_data

Data frame for which variable importances should be calculated.

fit

Caret fit.

formula_string

Formula string used in caret fit (or for creation of matrix used).

sparse

Use sparse matrix for caret fit if TRUE. If NULL then no matrix is created and df_data is passed.

metric

Metric to compare for feature importance. All metrics returned by performance_summary can be used.

features

Names of features to be calculated. If NULL all features are used.

b_sample

Named vector with priori probabilities of sampled data.

b_all

Named vector with priori probabilities of all data.

Value

Data frame with columns feature, importance, importance_cum (cumulated importance) and importance_sumnormed (importance normed by the sum of all importances).

See Also

Other variable importance functions: plot_varimp_cum, plot_varimp
