encode {hmsPM}R Documentation

Encode Feature Values

Description

Encode categorical feature with encoding generated by cate_encoding.

Usage

encode(x, encoding)

Arguments

x

Vector of categorical feature values.

encoding

Encoding to be applied as generated by cate_encoding.

Value

Encoded feature values as factor.

See Also

Other feature encoding functions: cate_encoding

Examples

(encoding = cate_encoding(letters[c(1:3,2:3,3)], method = "count"))
encode(letters[1:5], encoding)
