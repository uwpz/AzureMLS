plot_partial_dependency_cate {hmsPM}R Documentation

Plot partial dependency of categorical features

Description

Plot partial dependency of categorical features

Usage

plot_partial_dependency_cate(df_partialdep, df_data, feature_name,
  target_name, individual_flag_name = NULL, ylim, colors, min_width,
  print_refs = TRUE)

Arguments

df_partialdep

Data frame with structure as created by calc_partial_dependencies.

df_data

Feature and target data.

feature_name

Feature name.

target_name

Target name.

individual_flag_name

Optional column in df_data comprising 0 or 1 and which marks individual records to be highlighted in plots, i.e. black border or black dot.

ylim

Y-axis limits. See also lims.

colors

Colors for target levels.

min_width

Minimum width of bars for categorical variables.

print_refs

Print reference lines.

Value

ggplot2 object.
