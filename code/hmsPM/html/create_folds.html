Create Folds for Cross Validation

Description

Create Folds for Cross Validation

Usage

create_folds(n, fold_labels, n_folds = 5, n_repeat = 1, p = 0.8,
  method = "cv")

Arguments

n

Number of datasets.

fold_labels

Labels for datasets with values train or test. Method: 'byfold'.

n_folds

Number of folds.

n_repeat

Number of repetitions. Only used for method cv and byfold. Methods: 'cv', 'byfold'.

p

Probability or ratio of samples. Method: 'random'.

method

Fold type. Options: cv, byfold, random.

Value

List with train and test indices.
