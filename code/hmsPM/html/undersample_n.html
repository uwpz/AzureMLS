undersample_n {hmsPM}R Documentation

Undersample target

Description

Undersample (multi-)class target.

Usage

undersample_n(df_data, target_name = "target", n_max_per_level)

Arguments

df_data

Data frame with feature and target columns.

target_name

Name of target factor variable.

n_max_per_level

Maximum number of samples per target level.

Value

List with 'data' of undersampled target and base ratios b_sample and b_all.

See Also

Other prediction utility functions: detect_caret_predict_type, model_matrix, performance_summary, scale_predictions

Examples

library(magrittr)
df_data = data.frame(
  "target" = sample(10),
  "f1"     = factor(sample(letters[1:10])),
  "f2"     = sample(10)
)
c(df_data, b_sample, b_all) %<-% undersample_n(df_data, n_max_per_level = 2)
df_data
b_sample
b_all
