cate_encoding.target {hmsPM}R Documentation

Target Encoding

Description

The feature values are encoded with the count-encoded mean of the target reference level y_ref_level over each feature value.

Usage

cate_encoding.target(x, y, y_ref_level, decreasing = TRUE,
  enc_nas = TRUE, enc_others = TRUE)

Arguments

x

Feature values to be encoded.

y

Target values.

y_ref_level

Target reference level.

decreasing

Sort order of encoded values.

enc_nas

(Logical) Encode missing values? Default = TRUE. If TRUE, set NAs to 0.

enc_others

(Logical) Encode unkown/other values? Default = TRUE. If TRUE, set OTHERS to max(encoding)+1.

Value

Named vector with encoding for feature values.
