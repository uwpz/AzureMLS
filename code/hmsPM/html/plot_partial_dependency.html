plot_partial_dependency {hmsPM}R Documentation

Plot partial dependency

Description

Plot partial dependency

Usage

plot_partial_dependency(df_partialdep, df_data, feature_name,
  target_name = "target", individual_flag_name = NULL, ylim = NULL,
  colors = NULL, min_width = 0.2, show_legend = TRUE,
  df_partialdep_cv = NULL, add_cv_lines = FALSE,
  add_cv_conf_int = FALSE)

Arguments

df_partialdep

Data frame with structure as created by calc_partial_dependencies.

df_data

Feature and target data.

feature_name

Feature name.

target_name

Target name.

ylim

Y-axis limits. See also lims.

colors

Colors for target levels.

min_width

Minimum width of bars for categorical variables. Only relevant for (multi-)class targets.

show_legend

(Logical) Show legend?

df_partialdep_cv

(Optional) Data frame with structure as created by calc_partial_dependencies but with additional column run distinguishing cross-validation runs. Needs to be passed if either add_cv_lines or add_cv_conf_int is TRUE.

add_cv_lines

(Logical) Add lines per cross validation run to plot?

add_cv_conf_int

(Logical) Add confidence intervals derived from cross validation?

Value

ggplot2 object.
