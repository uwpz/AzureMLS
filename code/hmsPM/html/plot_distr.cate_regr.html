plot_distr.cate_regr {hmsPM}R Documentation

Plot categorical distribution on regression target

Description

Plot categorical distribution on regression target

Usage

plot_distr.cate_regr(df_data, feature_name, target_name, varimps = NULL,
  colors, ylim = NULL, show_legend = TRUE, varimp_digits = 2,
  verbose = TRUE)

Arguments

df_data

Data frame with columns named according to the values of the 'feature_name' and 'target_name' parameters.

feature_name

Feature name.

target_name

Target name.

varimps

Named vector with variable importances with an entry for the value of the 'feature_name' parameter.

colors

NOT USED

ylim

Y-axis limits. See also lims.

show_legend

(Logical) Show legend?

varimp_digits

Number of digits for displayed variable importance

verbose

(Logical) Show messages?

Value

Single ggplot.

See Also

Other distribution plots: plot_distr.cate_class, plot_distr.cate_multiclass, plot_distr.metr_class, plot_distr.metr_multiclass, plot_distr.metr_regr, plot_distr
