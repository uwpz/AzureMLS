model_matrix {hmsPM}R Documentation

Create Model Matrix

Description

Create a dense or sparse model/design matrix.

Usage

model_matrix(formula, data, sparse = FALSE)

Arguments

formula

Formula to use in caret fit (or for creation of matrix used).

data

Data frame with feature columns corresponding to formula.

sparse

(Logical) Create sparse model matrix?

Value

Model/design matrix (dense or sparse).

sparse.model.matrix, model.matrix
