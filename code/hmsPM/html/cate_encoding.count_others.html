cate_encoding.count_others {hmsPM}R Documentation

Count Encoding of Other Values

Description

Count encoding for remaining levels of self_topn-encoding. All n_top_levels are encoded to the value n_levels-n_top_levels+1, all remaining levels are numerated (count encoded) according to their frequency in (by default) decreasing order starting at 1.

Usage

cate_encoding.count_others(x, n_top_levels = 20, decreasing = TRUE,
  enc_nas = TRUE)

Arguments

x

Feature values to be encoded.

n_top_levels

Maximum number of levels.

decreasing

Sort order of encoded values.

enc_nas

(Logical) Encode missing values? Default = TRUE. If TRUE, set NAs to 0.

Value

Named vector with encoding for feature values.

Examples

cate_encoding(letters[c(1:3,2:3,3)], method = "count_others", n_top_levels = 2)
