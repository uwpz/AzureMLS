plot_residuals {hmsPM}R Documentation

Residual plot

Description

Two sided scatter plot used for residual plots.

Usage

plot_residuals(x, res, title = NULL, xlab = NULL, ylab = NULL,
  gradcol = NULL, ylim = NULL)

Arguments

x

Vector of values to be plotted on x-axis.

res

Vector of residual values.

title

Plot title.

xlab

Label of x-axis. Example: expression(hat(y)).

ylab

Label of y-axis. Example: expression(paste(y - hat(y))).

gradcol

Gradient colors. Used in regression residual plot.

ylim

Y-axis limits. See also lims.

Value

ggplot2 object.
