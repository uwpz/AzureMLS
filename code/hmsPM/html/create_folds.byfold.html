create_folds.byfold {hmsPM}R Documentation

Create Folds by Fold Label

Description

Filter train folds on "training data", test folds on "test data".

Usage

create_folds.byfold(n, fold_labels, n_folds, n_repeat)

Arguments

n

Number of datasets.

fold_labels

Labels for datasets with values train or test.

n_folds

Number of folds.

n_repeat

Number of repetitions.

Value

List with train and test indices.
