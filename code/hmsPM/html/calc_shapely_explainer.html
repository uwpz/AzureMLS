calc_shapely_explainer {hmsPM}R Documentation

Calculate Shapely based Explainer Data

Description

Creates help dataset which can later be used to create shapely values

Usage

calc_shapely_explainer(formula_string, df_train, n_permutations = 1000)

Arguments

formula_string

Formula string used in caret fit (or for creation of matrix used). Should contain no intercept.

df_train

Data used to train caret fit.

n_permutations

Number of permutations.

Value

Shapely explainer data.
