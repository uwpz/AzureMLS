stat_mode {hmsPM}R Documentation

Statistical mode

Description

Element appearing most often.

Usage

stat_mode(x, na.rm = TRUE)

Arguments

x

Vector of elements.

na.rm

Logical whether missing values should be removed.

Value

Element appearing most often.

See Also

Other data processing functions: convert_scale, impute, level_ratio, levels_to_extract, missing_ratio, winsorize

Examples

stat_mode(c(1, 2, 2, 3, 3, 3, NA))
stat_mode(c('a', 'b', 'b', NA))
stat_mode(factor(c('a', 'b', 'b', NA)))
