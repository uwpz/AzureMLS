calc_xgb_explainer {hmsPM}R Documentation

Calculate XGBoost Explainer Data

Description

Parses all xgboost trees and returns data frame needed for deriving explainer weights for predictions.

Usage

calc_xgb_explainer(fit, formula_string, sparse = TRUE, df_train)

Arguments

fit

Xgb caret fit.

formula_string

Formula string used in caret fit (or for creation of matrix used). Should contain no intercept.

sparse

Use sparse matrix for caret fit if TRUE.

df_train

Data used to train caret fit.

Value

Xgb explainer data.

See Also

xgboost

Other xgb explainer functions: calc_shapely_weights, calc_xgb_weights, plot_all_xgb_explanations, plot_xgb_explanations
