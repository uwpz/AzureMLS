Impute missing values

Description

Impute missing values

Usage

impute(x, strategy)

Arguments

x

Vector of values.

strategy

Imputation strategy. Options: 'random', 'median', 'mode', 'mean', 'zero'.

Value

Vector without missing values.

See Also

Other data processing functions: convert_scale, level_ratio, levels_to_extract, missing_ratio, stat_mode, winsorize

Examples

impute(c(0, 1, NA, 3, NA, 5), 'mean')
