cate_encoding.count {hmsPM}R Documentation

Count Encoding

Description

The feature values are numerated according to their frequency in (by default) decreasing order starting at 1.

Usage

cate_encoding.count(x, decreasing = TRUE, enc_nas = TRUE,
  enc_others = TRUE)

Arguments

x

Feature values to be encoded.

decreasing

Sort order of encoded values.

enc_nas

(Logical) Encode missing values? Default = TRUE. If TRUE, set NAs to 0.

enc_others

(Logical) Encode unkown/other values? Default = TRUE. If TRUE, set OTHERS to max(encoding)+1.

Value

Named vector with encoding for feature values.

Examples

cate_encoding(letters[c(1:3,2:3,3)], method = "count")
