detect_feature_scale {hmsPM}R Documentation

Detect feature scale

Description

metr

numeric data

ordi

factor data with order levels (numbers, letters, or LETTERS)

nomi

factor data with unordered levels or character data

Usage

detect_feature_scale(x)

Arguments

x

Vector of values.

Value

Scale string: metr, ordi, or nomi.

See Also

Other data check functions: check_feature_scale, check_same_feature_scales, check_target_type, detect_target_type

Examples

detect_feature_scale(1:3)
detect_feature_scale(letters[1:3])
detect_feature_scale(factor(letters[1:3]))
df_test = data.frame(list("f_metr" = 1:3, "f_nomi" = letters[1:3]))
purrr::map(df_test, detect_feature_scale)
