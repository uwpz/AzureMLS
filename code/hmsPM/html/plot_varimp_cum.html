plot_varimp_cum {hmsPM}R Documentation

Plot cumulative feature importances

Description

Bar plot of (top) cumulative feature importances in decreasing order.

Usage

plot_varimp_cum(df_varimp, n_top_features = NULL, colors = c("blue",
  "orange", "red"))

Arguments

df_varimp

Data frame with structure as created by calc_varimp_by_permutation

n_top_features

Number of top features to be plotted.

colors

Colors to use for grouping by color variable.

Value

ggplot2 object.

See Also

Other variable importance functions: calc_varimp_by_permutation, plot_varimp
