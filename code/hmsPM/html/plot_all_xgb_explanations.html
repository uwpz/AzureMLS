plot_all_xgb_explanations {hmsPM}R Documentation

Plot XGBoost Explanations for All Entries

Description

Plot XGBoost explanation plots for all entries in df_weights.

Usage

plot_all_xgb_explanations(df_explanations, id_name = "id",
  target_name = "target", ylim = c(0.01, 0.99),
  colors = alpha(c("blue", "red"), 0.5), verbose = TRUE)

Arguments

df_explanations

Data frame with weights to be plotted. Must have structure as returned by calc_xgb_weights.

id_name

Name of id variable in test df_explanations (required).

target_name

Name of target variable.

ylim

Y-axis limits. See also lims.

colors

Colors of fill bars.

verbose

(Logical) Show messages?

Value

List of ggplots.

See Also

xgboost

Other xgb explainer functions: calc_shapely_weights, calc_xgb_explainer, calc_xgb_weights, plot_xgb_explanations
