calc_gengap {hmsPM}R Documentation

Calcualate Generalization Gap

Description

Calcualate Generalization Gap

Usage

calc_gengap(df_data, formula_string, sparse = TRUE, method, tune_grid,
  cluster = NULL)

Arguments

df_data

Data frame for caret fit. Must contain variable "fold" with members "train" and "test".

formula_string

Formula string to use in caret fit (or for creation of matrix used).

sparse

Use sparse matrix for caret fit if TRUE. If NULL then no matrix is created and df_data is passed.

method

Method to use in caret fit.

tune_grid

Tune grid to use in caret fit

cluster

Name of cluster (for parallel processing).

Value

Data frame with performance metrics for train and test data.
