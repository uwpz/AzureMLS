performance_summary {hmsPM}R Documentation

Custom prediction performance function

Description

Custom summary function analog to caret defaultSummary.

Usage

performance_summary(data, lev = NULL, model = NULL)

Arguments

data

Data frame. TODO

lev

Vector of target levels. NULL in case of regression.

model

Model name as used for train.

Details

(Multi-)Classification performance metrics:

  • AUC

  • Weighted AUC

Regression performance metrics:

  • RMSE

  • spearman

  • pearson

  • ratioM

  • IqrE

  • AUC

  • MAE

  • MdAE

  • MAPE

  • MdAPE

  • sMAPE

  • sMdAPE

  • MRAE

  • MdRAE

Value

Named list with prediction performance metrics.

Other prediction utility functions: detect_caret_predict_type, model_matrix, scale_predictions, undersample_n
