plot_all_individual_dependencies {hmsPM}R Documentation

Plot all individual dependencies (of one observation)

Description

Plot list of all individual dependencies

Usage

plot_all_individual_dependencies(df_individualdep, df_data, feature_names,
  target_name = "target", individual_flag_name = "individual_flag",
  ylim = NULL, colors = NULL, min_width = 0.2,
  show_1_legend = TRUE, verbose = TRUE)

Arguments

df_individualdep

Data frame with structure as created by calc_individual_dependencies filtered to just one "id".

df_data

Feature and target data.

feature_names

Names of features to plot.

target_name

Target name.

individual_flag_name

Optional column in df_data comprising 0 or 1 and which marks individual records to be highlighted in plots, i.e. black border or black dot.

ylim

Y-axis limits. See also lims.

colors

Colors for target levels. Used in (multi-)class performance plots.

min_width

Minimum width of bars for categorical variables. Only relevant for (multi-)class targets.

show_1_legend

Show legend only on first plot.

verbose

(Logical) Show messages?

Value

List with all individual dependency plots.
