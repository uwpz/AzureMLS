convert_scale {hmsPM}R Documentation

Convert scale level

Description

Convert scale level

Usage

convert_scale(x, scale, name = NULL, verbose = F)

Arguments

x

Vector of values.

scale

Target scales. Options: "metr", "ordi", "nomi".

name

Variable name.

verbose

(Logical) Show messages?

Value

Converted vector of values.

See Also

Other data processing functions: impute, level_ratio, levels_to_extract, missing_ratio, stat_mode, winsorize

Examples

convert_scale(1:3, "metr")
convert_scale(1:3, "ordi")
convert_scale(1:3, "nomi")
# won't work
## Not run: convert_scale(letters[1:3], "metr")
convert_scale(letters[1:3], "ordi")
convert_scale(letters[1:3], "nomi")
# won't work
## Not run: convert_scale(factor(letters[1:3]), "metr")
convert_scale(factor(letters[1:3]), "ordi")
convert_scale(factor(letters[1:3]), "nomi")
