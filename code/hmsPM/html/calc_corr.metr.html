calc_corr.metr {hmsPM}R Documentation

Calculate Correlation Matrix

Description

Calculate Correlation Matrix

Usage

calc_corr.metr(df, cutoff, method, show_miss_ratio = TRUE,
  miss_ratio_prefix = "NA: ")

Arguments

df

Data frame with metric variables.

cutoff

Lower cutoff level. Value in [0,1]. Remove rows and columns in which no correlation value is above the cutoff value.

method

Correlation method. Options: 'spearman' or 'pearson'

show_miss_ratio

(Logical) Show missing ratio after variable name?

miss_ratio_prefix

Prefix for missing ratios.

Value

Correlation matrix.
