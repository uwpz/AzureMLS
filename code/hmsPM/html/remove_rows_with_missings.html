remove_rows_with_missings {hmsPM}R Documentation

Remove missings from XY

Description

Remove missings from feature and target values.

Usage

remove_rows_with_missings(df, verbose = TRUE)

Arguments

df

Data frame with feature and target values.

verbose

(Logical) Show messages?

Value

Data frame without missings in rows.

Examples

x = c(1, 2, NA, 4)
y = c(1, NA, 3, 4)
(df = data.frame(x, y))
(df_clean = remove_rows_with_missings(df))
